# Cost-Cutting Direction and Planning AI

![Project Banner](https://github.com/user-attachments/assets/5d4533d2-6ac3-4597-8b72-2ce57bbef29d)

## AI-powered cost optimization for businesses—track, analyze, and reduce expenses efficiently.

This project was developed for DUHacks 4.0 hackathon.

## 🚀 Overview

The *Cost-Cutting Direction and Planning AI* is a comprehensive expense management solution that leverages artificial intelligence to help businesses optimize their financial operations. Our platform analyzes financial data, identifies inefficiencies, and provides actionable recommendations to reduce costs and improve profitability.

### The Problem We Solve

Businesses often struggle with financial inefficiencies, leading to unnecessary expenses and suboptimal resource allocation. Our AI-driven solution helps businesses:

- *Analyze financial data* to detect overspending
- *Provide actionable cost-saving recommendations* based on AI insights
- *Optimize resource allocation* to reduce waste
- *Streamline workflows* and suggest budget-friendly alternatives
- *Support real-time tracking, forecasting, and financial planning*

With automated insights and interactive dashboards, our solution ensures businesses make *data-driven financial decisions* and *improve profitability*.

## ✨ Features

- *Full Expense Tracker* – Monitor all expenses in one place
- *Business Expense Tracker* – Manage business-related costs efficiently
- *Personal Expense Tracker* – Track personal spending and savings
- *Daily Expense Tracker* – Get daily spending insights and trends
- *Other Expenses* – Categorize and manage miscellaneous expenses

2. *Graph Plotting* – Visualizing cost patterns effectively for better decision-making
3. *Deploying AI Model* – Integrating Cohere API for real-time insights and seamless backend processing

We tackled these challenges by refining data preprocessing, optimizing API integration, and using *React Framer Motion* for smooth UI interactions.

## 🚀 Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm or yarn
- MongoDB account

### Installation

1. Clone the repository:
   bash
   git clone https://github.com/PanatiNitesh/DUHACKS.git
   cd DUHACKS
   

2. Install dependencies for frontend:
   bash
   cd frontend
   npm install
   

3. Install dependencies for backend:
   bash
   cd ../backend
   npm install
   

4. Create .env file in the backend directory and add the following:
   
   MONGODB_URI=your_mongodb_connection_string
   COHERE_API_KEY=your_cohere_api_key
   PORT=5000
   

5. Start the backend server:
   bash
   npm start
   

6. Start the frontend development server:
   bash
   cd ../frontend
   npm run dev
   

7. Open your browser and navigate to http://localhost:5173 to view the application.
